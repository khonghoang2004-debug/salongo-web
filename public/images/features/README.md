# Ảnh cho trang Tính năng

Copy 5 file ảnh vào **public/images/** hoặc **public/images/features/**.

| Tên file (đặt đúng) | Block trên web |
|---------------------|----------------|
| `anh-lich-hen.jpg` | Lịch hẹn thông minh |
| `anh-tinh-tien.jpg` | Thanh toán nhanh chóng |
| `anh-doanh-thu.jpg` | Báo cáo & phân tích chính xác |
| `dat-lich-online.jpg` | Trang đặt lịch online |
| `trang-nhan-vien.jpg` | Tài khoản riêng cho nhân viên |

- Định dạng: **.jpg** (hoặc đổi code sang .png nếu bạn dùng PNG).
- Nếu ảnh của bạn đang ở ngoài project (Desktop, thư mục features khác): copy vào đây và đổi tên cho đúng bảng trên.
