# Logo SalonGo

Logo hiện dùng ở **header** và **footer** lấy từ file:

**`salongo-logo.svg`** (file này, cùng thư mục)

Để đổi sang logo SVG của bạn:

1. Mở file logo SVG của bạn (đúng bản bạn muốn dùng).
2. Copy toàn bộ nội dung file đó.
3. Mở file **`salongo-logo.svg`** trong thư mục `public/` của project.
4. Xóa hết nội dung cũ và dán nội dung logo của bạn vào, rồi lưu file.

Hoặc: copy/ghi đè file logo của bạn lên `public/salongo-logo.svg` (đặt tên đúng là `salongo-logo.svg`).

Sau khi lưu, tải lại trang web (nếu vẫn thấy logo cũ thì thử Ctrl+F5 hoặc xóa cache trình duyệt).
